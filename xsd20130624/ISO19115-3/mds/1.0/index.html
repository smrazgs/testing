Metadata for Data and Services (MDS) Version: 1.0

Description

MDS 1.0 is an XML Schema implementation derived from ISO Geographic Information - Metadata - Part 1: Fundamentals. It includes wrapper namespace to support Catalog Service implementations. The XML schema was encoded using the encoding rules described in ISO/TS 19139:2007, Clause 8.

NOTE: The namespace URI of Metadata for Data and Services (MDS) Version: 1.0 doesn't reflect the version number of the XML Schema Implementation. The policy of ISO/TC 211 has evolved in order to reflect the versioning of the XML Schema Implementation as part of the namespace URI.

XML Schema of mds 1.0

mds.xsd is the root XML Schema document, i.e., the XML Schema document to be referenced by XML documents containing XML elements in the mds 1.0 namespace or by XML Schema documents importing the mds 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the mds namespace, but it does not contain the declaration of any types.

NOTE: The XML Schemca for mds 1.0 are available here as part of a zip archive regrouping all the XML Schema Implementations defined in ISO/TS 19139:2007. They are also available for remote access on the ISO ITTF Repository.

Schematron Validation Rules for mds 1.0

Schematron rules for validating instance documents of the mds 1.0 namespace are in mds.sch Other schematron rule sets that are required for a complete validation are: cit.sch, gex.sch, lan.sch, mas.sch, mcc.sch, mco.sch, mdb.sch, mmi.sch, mpc.sch, mrc.sch, mrd.sch, mri.sch, mrl.sch, mrs.sch, msr.sch, and srv.sch
