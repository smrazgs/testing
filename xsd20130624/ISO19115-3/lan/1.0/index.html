LANguage localization (LAN) Version: 1.0

Description

LAN 1.0 is an XML Schema implementation derived from ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals. It includes elements for cultural and linguistic adapatiblity. The XML schema was encoded using the encoding rules described in ISO/TS 19139:2007, Clause 8.

NOTE: The namespace URI of LANguage localization (LAN) Version: 1.0 doesn't reflect the version number of the XML Schema Implementation. The policy of ISO/TC 211 has evolved in order to reflect the versioning of the XML Schema Implementation as part of the namespace URI.

XML Schema of lan 1.0

lan.xsd is the root XML Schema document, i.e., the XML Schema document to be referenced by XML documents containing XML elements in the lan 1.0 namespace or by XML Schema documents importing the lan 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the lan namespace, but it does not contain the declaration of any types.

NOTE: The XML Schemca for lan 1.0 are available here as part of a zip archive regrouping all the XML Schema Implementations defined in ISO/TS 19139:2007. They are also available for remote access on the ISO ITTF Repository.

Other XML Schema for lan 1.0

language.xsd implements the UML conceptual schema defined in , 6.6.. It was created using the encoding rules defined in ISO 19118 and extended in ISO 19139 and contains the following classes: CountryCode, LanguageCode, LocalisedCharacterString, MD_CharacterSetCode, PT_FreeText, PT_Locale, and PT_LocaleContainer
