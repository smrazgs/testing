Metadata for Data Transfer (MDT) Version: 1.0

Description

MDT 1.0 is an XML Schema implementation derived from ISO 19115-1, Geographic Information - Metadata - Part 1: Fundamentals. The XML schema was encoded using the encoding rules described in ISO/TS 19139:2007, Clause 8.

NOTE: The namespace URI of Metadata for Data Transfer (MDT) Version: 1.0 doesn't reflect the version number of the XML Schema Implementation. The policy of ISO/TC 211 has evolved in order to reflect the versioning of the XML Schema Implementation as part of the namespace URI.

XML Schema of mdt 1.0

mdt.xsd is the root XML Schema document, i.e., the XML Schema document to be referenced by XML documents containing XML elements in the mdt 1.0 namespace or by XML Schema documents importing the mdt 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the mdt namespace, but it does not contain the declaration of any types.

NOTE: The XML Schemca for mdt 1.0 are available here as part of a zip archive regrouping all the XML Schema Implementations defined in ISO/TS 19139:2007. They are also available for remote access on the ISO ITTF Repository.

Other XML Schema for mdt 1.0

metadataTransfer.xsd implements the UML conceptual schema defined in , 6.5.2. It was created using the encoding rules defined in ISO 19118 and extended in ISO 19139 and contains the following classes: MX_Aggregate, MX_DataFile, MX_DataSet, AbstractMX_File, MX_ScopeCode, and MX_SupportFile
